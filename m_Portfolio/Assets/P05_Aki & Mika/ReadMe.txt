・Sceneデータ「DEMO_MagicaCloth」を動作させるには、アセット「Magica Cloth」が必要になります。
また、「Magica Cloth（マギカ・クロス）」を動作させるにはUnity Job System + Burstコンパイラが必要になります。
アセットストア
https://assetstore.unity.com/packages/tools/physics/magica-cloth-160144

・Sceneデータ「DEMO_DynamicBone」を動作させるには、アセット「Dynamic Bone」が必要になります。
アセットストア：
https://assetstore.unity.com/packages/tools/animation/dynamic-bone-16743

・当モデルではより自然な関節可動を行うために各所にボーンコンストレイトを使用しております。
改変をされる場合にはコンストレイトの再設定にご注意ください。

・各衣裳の着脱の際には貫通防止のシェイプキーを各所に設定しておりますのでご注意ください。
また、場合により部分的に物理シミュレーションをオフにする必要があります。各該当衣装のプレファブを参照してください。

・ヒューマノイドキャラクターのマテリアルはフォルダ「Mterials」内の
パッケージデータ「URPconvert」を展開することでURP対応マテリアルへと上書き変換できます。

※URPプロジェクトのデフォルトのスキンウェイトは2ボーンですが、この製品では4ボーンを推奨しています。
プロジェクト設定で4ボーンに変更してください。 これは、各スキンメッシュのインスペクター内ではなく、プロジェクト設定で実行する必要があります。


-The asset "Magica Cloth" is required to operate the Scene data "DEMO_Magica Cloth".
In addition, the Unity Job System + Burst compiler is required to operate "Magica Cloth".
Asset store
https://assetstore.unity.com/packages/tools/physics/magica-cloth-160144

-The asset "Dynamic Bone" is required to operate the Scene data "DEMO_Dynamic Bone".
Asset store:
https://assetstore.unity.com/packages/tools/animation/dynamic-bone-16743

-In this model, bone constraints are used in various places to perform more natural joint movement.
Please be careful about resetting the constraint when making modifications.

-Please note that shape keys to prevent penetration are set in various places when putting on and taking off each costume.
You may also need to partially turn off physics simulation. Please refer to the prefab of each applicable costume.

-Humanoid character materials are in the folder "Mterials"
By expanding the package data "URP convert", it can be overwritten and converted to URP compatible material.

-The default skin weight for URP projects is 2 bones, but this product recommends 4 bones.
Please change to 4 bones in the project setting. It must be done in the project setting, not in the inspector of each skin mesh.
